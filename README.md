# survival
Survival Data Analysis using advanced statistical methods, including Kaplan–Meier estimation, Cox proportional hazards modeling, and time-dependent variable analysis, implemented in Stata and R.

Survival Data Analysis

This repository demonstrates applied survival analysis using classical and extended Cox proportional hazards models in Stata and R.

Topics Covered

Kaplan–Meier survival estimation

Log-rank tests

Cox proportional hazards regression

Proportional hazards (PH) assumption testing

Time-dependent covariate analysis

Extended Cox models

Immortal time bias

Datasets

Australian Methadone Study (ADDICTS)

Stanford Heart Transplant Study (HEART)

Software

Stata (stcox, tvc(), stsplit)

R (survival, coxph, cox.zph)
