# Survival Data Analysis
## Time-dependent covariates and extended Cox models

Applied **survival data analysis** using advanced statistical methods, including:

- Kaplan–Meier estimation  
- Cox proportional hazards (PH) modeling  
- PH assumption diagnostics  
- Time-dependent covariates / extended Cox models  
- Immortal time bias considerations  

Implemented in **Stata** and **R**.

 * Created by: Ali Mirzazadeh
 * For questions or assistance: ali.mirzazadeh@ucsf.edu
---

## Overview

This repository demonstrates applied survival analysis using classical and extended Cox proportional hazards models in **Stata** and **R**, with worked examples on well-known survival datasets.

---

## Topics covered

- **Kaplan–Meier** survival estimation  
- **Log-rank** tests  
- **Cox proportional hazards** regression  
- **PH assumption** testing  
- **Time-dependent covariate** analysis  
- **Extended Cox** models  
- **Immortal time bias**
